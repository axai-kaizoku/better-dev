## Commands - info

- [ ] ipconfig getifaddr en0 -- get ipaddress of your device
- [ ] ifconfig | grep "inet " -- get all the network interfaces of your device
- [ ] ifconfig -- to get all the brigdes in detail
- [ ] networksetup -listallhardwareports -- lists all the hardware ports associated with your device and network
- [ ] if config -- interface management; ipconfig -- ip configuration information
- [ ] netstat -rn -- routing tables, destination -> gateway
- [ ] scutil --dns -- dns configuration
- [ ] lsof -i -- current processes of network connection
- [ ] ip address changes on wifi connection
- [ ] route -n get default | grep gateway -- get wifi routers ip address
- [ ] arp -a -- get devices connected to wifi router; ip address . mac address
- [ ] ping -c 1 192.168.0.12 -- ping the device connected to same wifi
- [ ] nslookup 192.168.0.12 -- get mac address
- [ ] host 192.168.0.12 -- ^
- [ ] nmap -- brew package to get host name (device name) of connected devices
- [ ] http://192.168.0.1 -- goes to router admin page
